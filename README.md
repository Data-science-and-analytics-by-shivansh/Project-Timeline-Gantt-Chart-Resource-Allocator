## Features (Most Advanced Version)
- Dynamic Gantt with conditional formatting for milestones, delays, critical path
- **VBA Drag-and-Drop** task adjustment + UserForm editor
- Automatic resource leveling (Solver + PuLP optimization)
- Burndown chart with Earned Value Management (SPI, CPI, EAC, ETC)
- Critical Path Method + Monte Carlo risk simulation (10,000 iterations)
- What-If scenarios for resource availability & crashing
- Power BI interactive dashboard with custom Gantt & forecasting
- SQL backend ready for enterprise scaling

## Technologies
Excel (VBA + Solver + Dynamic Arrays) • Power BI • Python (PuLP, Prophet, NumPy) • SQL
